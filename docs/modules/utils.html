stringWidth: { default: typeof stringWidth }
  • Defined in node_modules/string-width/index.d.ts:1

Type declaration

    • (string: string): number

    • Get the visual width of a string - the number of columns required to display it.

      Some Unicode characters are fullwidth and use double the normal width. ANSI escape codes are stripped and doesn't affect the width.

      example
      import stringWidth = require('string-width');

stringWidth('a');
//=> 1

stringWidth('');
//=> 2

stringWidth('\u001B[1m\u001B[22m');
//=> 2

      Parameters

      • string: string

      Returns number

stringLength

  • stringLength(string: string): number
    • Defined in node_modules/string-length/index.d.ts:20

    Get the real length of a string - by correctly counting astral symbols and ignoring ansi escape codes.

    String#length erroneously counts astral symbols as two characters.

    example
    import stringLength = require('string-length');

''.length;
//=> 2

stringLength('');
//=> 1

stringLength('\u001B[1municorn\u001B[22m');
//=> 7

    Parameters

    • string: string

    Returns number
